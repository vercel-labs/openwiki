# OpenWiki Docs

This folder tracks product and implementation intent for OpenWiki.

- `active/` contains current plans, design notes, and unresolved decisions.
- `completed/` contains shipped or superseded plans that remain useful context.
- `harness-gaps.md` captures friction that should become better docs, tests, errors, or tooling.

If an active plan and the code disagree, either update the code to match the plan or land a separate docs update that explains the new direction.
